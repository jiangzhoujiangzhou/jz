<h2>ZAKER新闻APP</h2>

<h3>项目介绍</h3>
ZAKER，是2010年12月由广州坚和网络科技有限公司开发的一款阅读类软件。 [1]  主要功能：杂志排版、个性定制、社交互动、离线下载。
